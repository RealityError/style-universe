# Style Universe · 风格宇宙

> 150+ 设计风格全景库，集 2D 展示、3D 抽卡体验、后台管理于一体。

## 在线访问

🔗 **[在线体验](https://你的用户名.github.io/style-universe/)**

## 功能

### 🎨 2D 风格库 (`index.html`)
- 150+ 种设计风格，涵盖拟态系、发光系、复古系、艺术系、文化系等 17 大类
- 分类筛选、关键词搜索
- 点击查看风格详情，预览 UI 组件、交互组件、代码片段
- 导出 CSS / SCSS / Tailwind / JSON 配置

### 🎮 3D 抽卡体验 (`3d.html`)
- Three.js 驱动的 3D 沉浸式空间
- 82 种风格，每种有独特的 3D 艺术装置和粒子效果
- 10 种自定义 GLSL Shader 材质（流体、发光、极光、全息、故障、水面、火焰、水墨、晶体、矩阵）
- 卡牌翻转抽取 + SSR/SR/R/N 稀有度系统
- 相机运镜 + 过渡动画
- 收藏历史 + URL 直链预览

### 📊 后台管理 (`admin.html`)
- 全部风格的可视化管理面板
- 按稀有度、标签筛选排序
- 配色方案查看 + CSS Token 一键复制
- 直跳 3D 预览

## 技术栈

- **Three.js** (r160) — 3D 渲染
- **GLSL** — 自定义 Shader 材质
- **UnrealBloomPass** — 后处理辉光
- **纯 HTML/CSS/JS** — 零依赖，无需构建

## 本地运行

直接浏览器打开 `index.html` 即可。

如需 3D 页面的完整功能（ES Module），建议用本地服务器：

```bash
npx serve .
```
